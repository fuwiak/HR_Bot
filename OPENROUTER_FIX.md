# Исправление ошибки 404 OpenRouter API

## Проблема
Ошибка `404 Not Found` при запросе к OpenRouter API.

## Возможные причины

1. **Модель недоступна**: `x-ai/grok-4.1-fast:free` может быть недоступна или иметь другое имя
2. **Неправильный URL**: Проверьте правильность URL API
3. **Проблема с API ключом**: Убедитесь, что ключ правильный

## Решения

### 1. Проверьте доступные модели на OpenRouter
Перейдите на https://openrouter.ai/models и найдите доступные модели Grok.

### 2. Попробуйте альтернативные модели
Установите переменную окружения:
```bash
OPENROUTER_MODEL=x-ai/grok-beta
# или
OPENROUTER_MODEL=x-ai/grok-2-1212
# или
OPENROUTER_MODEL=grok-beta
```

### 3. Проверьте URL
Убедитесь, что URL правильный:
```bash
OPENROUTER_API_URL=https://openrouter.ai/api/v1/chat/completions
```

### 4. Проверьте API ключ
Убедитесь, что `OPENROUTER_API_KEY` установлен правильно.

## Тестирование

После исправления проверьте логи - должны появиться детальные сообщения об ошибках.
