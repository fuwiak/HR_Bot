# Email Manager — custom agent skill для AnythingLLM

Просмотр писем Yandex и отправка ответов прямо из чата AnythingLLM. Письма забираются через HR_Bot backend (IMAP Yandex).

## Требования

- Node.js 18+ (для `fetch`)
- AnythingLLM с поддержкой custom agent skills
- В окружении AnythingLLM задать **`HR_BOT_BACKEND_URL`** — URL бэкенда HR_Bot **без слэша в конце** (например: `https://backend-production-099b.up.railway.app`). Альтернативно: **`BACKEND_URL`**.

Бэкенд должен иметь доступ к почте Yandex (переменные `YANDEX_EMAIL`, `YANDEX_IMAP_PASSWORD` и т.д.) и реализовывать эндпоинты:
- `GET /api/email/recent?limit=N` — список писем
- `POST /api/email/reply` — отправка ответа (body: `to_email`, `subject`, `content`)

## Установка

1. Папка **`email-manager`** копируется в AnythingLLM при сборке (см. `frontend/Dockerfile`: копируется весь `plugins/agent-skills/`). На Railway после деплоя скилл появляется в **Agents → Skills** как «Email Manager».

2. В **Variables** сервиса **Frontend (AnythingLLM)** в Railway добавьте:
   - **`HR_BOT_BACKEND_URL`** = `https://ваш-бэкенд.up.railway.app` (без `/` в конце)

3. В AnythingLLM включите скилл **«Email Manager»** в настройках агента (workspace для ответов на письма).

4. Чтобы агент вызывал скилл при запросах про почту, в **System Prompt** агента можно добавить:
   ```
   Когда пользователь просит показать письма, последнее письмо, ответить на письмо — вызывай скилл "Email Manager" и передавай полный текст сообщения пользователя. Команды: mail письма, письмо 1, mail ответь на письмо 2: текст ответа.
   ```

## Команды (в чате AnythingLLM)

Все команды можно вводить с префиксом **mail** или сразу **письм**:

| Команда | Действие |
|--------|----------|
| **mail письма** / **письма** | Список последних писем (тема, от кого) |
| **письмо 1** / **mail письмо 1** | Показать полный текст письма №1 |
| **mail ответь на письмо 2: Ваш текст ответа** | Отправить ответ на письмо №2 |

Номер письма — это порядковый номер в списке (1 = самое новое письмо).

## Примеры

- Показать список писем:
  ```
  mail письма
  ```
  или
  ```
  письма
  ```

- Показать первое (последнее) письмо:
  ```
  письмо 1
  ```

- Ответить на второе письмо:
  ```
  mail ответь на письмо 2: Добрый день! Высылаю запрошенную информацию. С уважением.
  ```

## Где это работает

- **В чате AnythingLLM** (веб-интерфейс) — после включения скилла и настройки `HR_BOT_BACKEND_URL` вы можете писать «письма», «письмо 1», «ответь на письмо 2: …» и получать список писем, текст письма и отправлять ответы.
- Письма по-прежнему можно смотреть и обрабатывать в **Telegram** (команда `/email_check`, кнопка «Подготовить ответ» и т.д.).
