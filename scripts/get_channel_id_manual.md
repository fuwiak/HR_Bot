# Как получить ID канала Telegram @HRAI_ANovoselova_Лиды

## Способ 1: Через специальных ботов (самый простой)

1. Добавьте бота в канал @HRAI_ANovoselova_Лиды как администратора
2. Отправьте любое сообщение в канал
3. Используйте одного из ботов для получения ID:
   - @userinfobot - отправьте ему ссылку на канал
   - @getidsbot - добавьте его в канал и он покажет ID
   - @RawDataBot - добавьте его в канал

## Способ 2: Через веб-версию Telegram

1. Откройте канал @HRAI_ANovoselova_Лиды в веб-версии Telegram
2. Посмотрите URL в адресной строке - там будет ID канала
3. Формат: `https://web.telegram.org/k/#-1001234567890` (число после # - это ID)

## Способ 3: Через попытку отправки сообщения

1. Добавьте бота в канал как администратора
2. Попробуйте отправить сообщение через скрипт
3. Если ID неверный, Telegram вернет ошибку с правильным форматом

## Способ 4: Через API (если бот уже в канале)

Если бот уже добавлен в канал, можно использовать скрипт `get_channel_id.py`:

```bash
python scripts/get_channel_id.py @HRAI_ANovoselova_Лиды
```

## Важно!

- ID канала обычно начинается с `-100` (например: `-1001234567890`)
- Бот должен быть администратором канала для отправки сообщений
- Для приватных каналов бот должен иметь права на отправку сообщений
